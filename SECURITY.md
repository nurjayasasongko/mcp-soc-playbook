# Security Policy

## Scope

This repo documents MCP server configurations and SOC workflows. It includes references to offensive tools (Metasploit, SQLMap, etc.) for documentation purposes only.

## Responsible Use

- Use this repo only in environments you have explicit permission to test
- Never use offensive tool configurations against systems you don't own
- API keys and credentials should never be committed to this repo

## Reporting Issues

Found a security issue in this documentation or configs? Open an issue or contact via Medium.
